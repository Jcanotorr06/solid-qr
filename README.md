# solid-qr

[![pnpm](https://img.shields.io/badge/maintained%20with-pnpm-cc00ff.svg?style=for-the-badge&logo=pnpm)](https://pnpm.io/)

A QR code generator for SolidJS

> **Note** After using this template, you have to search and replace all `solid-qr` and similar strings
> with appropriate texts.
>
> `solid-qr` should be a **kebab-case** string representing the name of you monorepo.
>
> `A QR code generator for SolidJS` should be a **Normal case** string with the description of the repository.
>
> `Jcanotorr06` should be a **kebab-case** string from your profile URL.

## Quick start

Install it:

```bash
npm i solid-qr
# or
yarn add solid-qr
# or
pnpm add solid-qr
```

Use it:

```tsx
import solid-qr from 'solid-qr'
```
